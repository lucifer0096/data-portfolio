## How to navigate this repo

1. Start with the **Projects** table above and click into a folder of interest.  
2. Read the project `README.md` for context, dataset details, and screenshots/diagrams (where available).  
3. Open the associated Excel / Power BI / R files to explore the analysis, dashboards, or app code.  
4. For the Shiny app, follow the steps in the volcano project README to run it locally in R.
